# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates:

- All active branches

## Reporting Security Vulnerabilities

To report a security vulnerability, please create a [GitHub issue](https://github.com/lxqt/qps/issues) or contact us directly at security@lxqt-project.org. We will review all reports and take appropriate actions.

## Security Best Practices

- **Keep your dependencies updated:** Regularly update dependencies to benefit from the latest security patches.
- **Review code changes:** Always review code changes before merging them into the main branch.
- **Use secure configurations:** Ensure that applications are configured to follow security best practices, including setting proper permissions and using secrets management.
